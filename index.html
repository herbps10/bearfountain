Services

We like data.

One of the largest challenges in web design is creating intuitive methods to interact with data.

Web Design

We build original designs handcoded in standards compliant HTML and CSS. We can help you build a powerful web presence by designing a modern look for your website that fits your business.

Content Management

Content management systems allow you to be in complete control of the content on your website. We can set up your website with software such as Wordpress that allows you to keep your website updated yourself, no programming knowledge required.

Web Programming

We can give your website dynamic functionality with web programming technologies such as PHP, Ruby, and Javascript. We can even let your website take phone calls!

eCommerce

Enable your website to accept payments online with powerful online store software. We can set you up with either a preexisting eCommerce package such as Magento, or develop a custom solution that allows you to take payment through third party handlers such as PayPal or Authorize.net.
