Services

Custom Design

We build original designs handcoded in standards compliant XHTML and CSS. We can help you build a powerful web presence by designing a modern look for your website that fits your business.

We have a combined ten years of experience with web design. We have built websites for organizations, businesses, and individuals. This has included creating blogs and setting up e-commerce websites.

Web Programming

Our experience includes developing web applications using PHP and Ruby on the server side and dynamic frontends with Javascript. For data storage, our applications have used SQL databases as well as innovative non relational datastores such as Redis.
